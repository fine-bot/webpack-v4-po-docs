# 打包分析工具
## webpack-bundle-analyzer
`webpack-bundle-analyzer` 是打包分析神器，可以直观的显示打包文件的大小及其依赖，方便定位构建过程性能瓶颈。
### 安装
```sh
npm i -D webpack-bundle-analyzer
```

### 总结
使用 `webpack-bundle-analyzer` 可以轻松得到各模块的打包大小及其之间的依赖关系，从而对症下药进行优化。
